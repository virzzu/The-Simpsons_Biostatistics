# The-Simpsons_Biostatistics
Este repositorio es un proyecto de trabajo de la asignatura de Estadística y R del Máster en Bioinformática por la UNIR. Análisis integral de expresión génica en 59 pacientes (The Simpsons). Incluye pruebas de normalidad, reducción de dimensiones (PCA), Clustering, estadística descriptiva y modelos de regresión logística.
